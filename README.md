This project delivers a hybrid NLP pipeline that turns unstructured customer feedback into clear, decision-ready insights by combining probabilistic topic modelling (LDA via Gensim), transformer-based semantic clustering with BERTopic, and LLM-driven interpretation using phi-4-mini. LDA provides a transparent, auditable baseline, BERTopic captures deeper semantic patterns through embeddings and clustering, and the LLM layer translates topics into concise, business-relevant narratives. Together, this approach balances interpretability with modern NLP performance, enabling scalable analysis of customer sentiment and themes while reducing manual review effort and improving the quality and consistency of insights for product and operational decision-making.
